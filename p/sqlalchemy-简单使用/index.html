SQLAlchemy 简单使用

Jan 10, 2024
3 minute read

ORM

1

Object-Relational Mapping技术，实现的是把关系型数据库的表结构映射到对象上。

 1# Id	name
 2# 1		Alice
 3# 2		Bob
 4# 3		Tom
 5# 以上包含id和name列的User表，可以用以下类实例表示
 6
 7class User(object):
 8    def __init__(self, id, name, gender):
 9        self.id = id
10        self.name = name
11        self.gender = gender
12
13[
14    User('1', 'Alice', 'female'),
15    User('2', 'Bob', 'male'),
16    User('3', 'Tom', 'male')
17]

Python中，最有名的ORM框架是SQLAlchemy。

简单使用

 1from sqlalchemy import create_engine, Column, Integer, String
 2from sqlalchemy.orm import sessionmaker, declarative_base
 3import pymysql
 4
 5# 读取配置
 6import configparser
 7config = configparser.ConfigParser()
 8with open("config.conf", "r") as conf:
 9    config.read_file(conf)
10    username = config.get("mysql_db", "username")
11    password = config.get("mysql_db", "password")
12    host = config.get("mysql_db", "host")
13    port = config.get("mysql_db", "port")
14    databaseName = config.get("mysql_db", "databaseName")
15
16# 创建对象的基类
17Base = declarative_base()
18
19
20# 表类
21class User(Base):
22    # 表名
23    __tablename__ = 'user'
24
25    # 表结构
26    id = Column(Integer, primary_key=True)
27    name = Column(String(20))
28    gender = Column(String(20))
29
30
31# 初始化数据库连接
32#                       数据库类型 数据库驱动 用户名     密码      主机地址  端口    要连接的数据库名称
33engine = create_engine(f'mysql+pymysql://{username}:{password}@{host}:{port}/{databaseName}')
34
35# 创建Dbsession类
36DBsession = sessionmaker(bind=engine)
37# 创建session对象
38session = DBsession()
39# 创建User对象
40newUser = User(id="4", name="Lily", gender='Female')
41
42try:
43    # 新增对象实例记录
44    session.add(newUser)
45    # 提交到数据库（保存改动）
46    session.commit()
47except Exception as e:
48    print(e)
49
50# 关闭session
51session.close()

常见问题

‘Engine’ object has no attribute ’execute’

参见：AttributeError: ‘Engine’ object has no attribute ‘execute’ when trying to run sqlalchemy in python to manage my SQL database

1from sqlalchemy import create_engine, text
2import pymysql
3
4engine = create_engine("mysql+pymysql://{user}:{pw}@localhost:3306/{db}".format(user=user, pw=pw, db=db))
5
6results = engine.execute(text("SELECT * FROM companyname;"))
7
8# AttributeError: 'Engine' object has no attribute 'execute'

在sqlalchemy 2.0版本，取消了engine.execute()用法，转为了依赖connector的需要时才连接的用法。

1

<stdin>:1: RemovedIn20Warning: The Engine.execute() method is considered legacy as of the 1.x series of SQLAlchemy and will be removed in 2.0. All statement execution in SQLAlchemy 2.0 is performed by the Connection.execute() method of Connection, or in the ORM by the Session.execute() method of Session. (Background on SQLAlchemy 2.0 at: https://sqlalche.me/e/b8d9)

2.0版本以后，正确的使用方式为：

1with engine.connect() as conn:
2    result = conn.execute(query)
