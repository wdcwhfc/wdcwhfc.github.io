一、diff

在使用版本控制的编程工作中，diff是一个常用的功能。git diff和svn diff都能很直观地输出同一文件的不同版本之间的差异。差异的表现形式就是列出从原始文件到新版文件所需要的操作。

Diff输出结果的优劣取决于操作的次数和复杂度，怎么计算并输出最优结果呢？。

- 首先，操作的次数越少越好；
- 其次，`整块删除，整块新增`；最好能呼应代码结构来实现；

二、图搜索

以“ABCABBA”和“CBABAC”分别作为原始文本和目标文本为例，构建一张图，原始文本做横轴，目标文本做纵轴。

从坐标点(0, 0)开始，横向移动表示删除，纵向移动表示新增，对角线移动表示保留原内容，那么当到达坐标点(7, 6)时，表示从原始文本变成了目标文本。通过这种方式，寻找最少次数的diff方法就变成了寻找最短路径，diff最优解就是最直观的最短路径。

三、Myers算法

算法原理

  • 定义参数 d 和 k
    • d：路径的长度（操作的次数）
    • k：当前坐标 x-y 的值
  • 最优坐标：d、k 值固定的情况下，x 值最大（优先删除）
  • 基础结论：
    • d 的取值范围是：0～原始文本与目标文本的长度之和（多行文本则以行为单位）
    • k 的可能取值为：range(-d, d, 2)（头尾都包含）
  • 目标：在取值范围内，遍历 d 和 k 值，直到找到一对值，使得当前最优坐标落到目标坐标上。坐标。

算法实现

  • 遍历 d、k值，寻找最优坐标：
1for d in range(0, l1+l2+1, 1):  # d: step num
2    for k in range(-d, d+1, 2):  # k: x - y
3        if k == -d or (k != d and lastV[k-1] < lastV[k+1]):
4            x = lastV[k+1]
5        else:
6            x = lastV[k-1] + 1
  • 对角线
1while x < l1 and y < l2 and self.list_x[x].strip() == self.list_y[y].strip():  
2    x += 1
3    y += 1
  • 记录最优坐标
 1trace = {}  # key：d / value：{k:x}
 2
 3# print trace

 6    1: {-1: 0, 1: 1},

  • 回溯路径
 1for d in range(len(self.trace) - 1, 0, -1):
 2    k = x - y
 3    lastV = self.trace[d - 1]
 4
 5    if k == -d or (k != d and lastV[k - 1] < lastV[k + 1]):
 6        prev_k = k + 1
 7    else:
 8        prev_k = k - 1
 9
10    prev_x = lastV[prev_k]
11    prev_y = prev_x - prev_k
  • 记录每一步的操作
 1while prev_x < x and prev_y < y:
 2    self.operation.append(MOVE)
 3    x -= 1
 4    y -= 1

 6if prev_x == x:
