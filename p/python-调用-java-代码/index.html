0x01 准备jar包

Java代码示例：

 1// 需要public类型的类才可以被调用
 2public class Test {
 3    public static void main(String[] args) throws Exception {
 4        System.out.println("Hello, main");
 5    }
 6
 7    // 需要public类型的方法才可以被调用
 8    public String test(String str) throws Exception {
 9        System.out.println("[+] Method `test` is called.");
10        return str;
11    }
12}

生成jar包:

1
2
3
4

javac Test.java // 生成Test.class
jar cvf output.jar Test.class // 生成output.jar
// 此处需要注意，如果class文件在其他目录下，指定目录路径生成的jar包，
// 在被调用时，需要保持对应的路径关系；同意目录下直接指定class文件，则直接调用类名即可

Python代码示例:

 1import jpype
 2
 3# 指定jar文件路径或存放目录
 4jarpath = "output.jar"
 5# 启动JVM，第一个参数指定jvm的位置，也可以通过getDefaultJVMPath()函数获取系统默认jdk环境
 6# jpype.startJVM("/path/to/***libjvm.so", "-ea", f"-Djava.class.path={jarpath}")
 7jpype.startJVM(jpype.getDefaultJVMPath(), "-ea", f"-Djava.class.path={jarpath}")
 8Test = jpype.JClass("Test")
 9testObj = Test()
10print(testObj.test("Hello, JPype"))
11# 关闭JVM
12jpype.shutdownJVM()
