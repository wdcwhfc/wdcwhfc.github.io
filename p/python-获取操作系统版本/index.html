Python 获取操作系统版本

Nov 09, 2023
1 minute read

1、使用 os 模块获取操作系统类型

通过 os.name 可以获取当前操作系统的类型，当前只注册了 3 个值：

  • posix： 对应 linux
  • nt： 对应 windows
  • java： 对应 java 虚拟机

2、使用 sys 模块获取系统的平台标识

通过 sys.platform 可以获取当前系统的平台标识，返回的标识与系统的对应关系如下：

  • win32： 表示 windows 系统
  • linux： 表示 linux 系统
  • cygwin： 表示 Windows/Cygwin 系统
  • darwin： 表示 macOS 系统

3、使用 platform 模块获取更详细的信息

（1）通过 platform 模块提供的各种方法，可以获取当前环境的操作系统、网络、处理器等更详细的信息：

 1import platform   #导入platform模块
 2
 3print('操作系统名称：', platform.system()) #获取操作系统名称
 4print('操作系统名称及版本号：', platform.platform()) #获取操作系统名称及版本号
 5print('操作系统版本号：', platform.version()) #获取操作系统版本号
 6print('操作系统的位数：', platform.architecture()) #获取操作系统的位数
 7print('计算机类型：', platform.machine()) #计算机类型
 8print('计算机的网络名称：', platform.node()) #计算机的网络名称
 9print('计算机处理器信息：', platform.processor()) #计算机处理器信息
10print('包含上面所有的信息汇总：', platform.uname())#包含上面所有的信息汇总
11
12# 使用 platform 模块还可以获得计算机中 python 的一些信息：
13print('python版本：', platform.python_version())  #打印python版本
14print('python版本：', platform.python_version_tuple())  #打印python版本
15print('python实现方式：', platform.python_implementation())  #打印python实现方式
16print('python版本分支：', platform.python_branch())  #打印python版本分支
17print('python版本修订：', platform.python_revision())  #打印python版本修订
18print('python编译信息：', platform.python_build())  #打印python编译信息
19print('python编译器：', platform.python_compiler())  #打印python编译器
